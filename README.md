功能介绍：
供债券操盘手使用，需求如下：由于债券的交易量和成交频度和股票完全不同，所以希望通过计算机自动监控不同个债的买一卖一的挂单量或价格的变化，一旦发生变化就自动播放音乐预警，这样操盘手就不需要长时间呆在电脑前盯盘了（终于可以解放去做其它事了）。
----------
可执行jar文件在文件夹“jar\MyHttpFetcher_jar”中。
----------
使用方法：
0. 首先安装jdk。
1. 将当前文件夹都复制到计算机本地目录中。
2. 修改setting.txt，其中在两个"#”之间的值为设定值，设定内容包含：每次刷新间隔时间、所有个债个股代码文件所在路径、播放音乐文件所在路径，设定完毕后保存。
3. 修改productlist.txt文件，文件中填写6位债券代码（多个债券则填写多行）。
4. 完成设置后，双击“开始抓取.bat”开始运行程序。如发生债券的买一卖一的挂单量或价格发生变化时，会发出预警声音提醒。如需关闭，则将cmd界面关闭即可。
另：如出现如：“Exception in thread "main" java.io.FileNotFoundException: d:\Temp\MyHttpFetcher_
jar\productlist.txt (系统找不到指定的路径。)”的提示，就表示配置文件中的文件地址配置有误，计算机未找到这个文件，需重新配置。
